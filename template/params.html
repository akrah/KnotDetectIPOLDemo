You can first ajust different parameters associated to the image accumulation generation used in the knot detection algorithm: alpha resolution ( a ), the minimal and maximal radius values of the analyse interval (rMin, rMax) and the minimal z-motion to be considered ( zMin ), default 10 to remove background noise.

You can also change the parameter of the knot detection algorithm: the threshold value to binarized the image ( tBin ), the maximum distance between two connected components ( dmax ), set by default to 20 according the mean awaided with of the knot and finally the min size to consider a set of pixel as a knot ( minSize ).
